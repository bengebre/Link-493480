# Link-493480
Link detections of (493480) 2014 YZ<sub>49</sub> in TESS thresholded difference images.

The <a href="https://github.com/bengebre/Link-493480/blob/master/493480.ipynb">rendered notebook</a> loads 6 previously calculated TESS thresholded difference image frames and their corresponding observations times and then "links" candidate detections across those frames.
